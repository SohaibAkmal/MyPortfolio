About

The person behind the test suite.

I've spent five years being professionally skeptical. While developers ship with confidence, I show up asking: what could break here? That instinct — for edge cases, for failure modes, for the thing nobody tested — is what I bring to every project.

My career spans healthcare SaaS at MTBC-CareCloud, IoT device monitoring at MRS Technologies, enterprise automation at Elixir Technologies, and complex API systems at Synergy-IT. Each role has sharpened the same skill: seeing a system the way an adversary would, then making it bulletproof before anyone else encounters it.

I write automation that actually catches bugs — not just automation that runs green.
